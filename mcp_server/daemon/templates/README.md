# Service Configuration Templates

This directory contains **reference templates** for platform-specific service configurations.

## Important Note

**These templates are for reference only.** The actual service files are generated dynamically by the service managers with correct paths and settings.

When you run `graphiti-mcp daemon install`, the appropriate service manager will:
1. Detect your Python interpreter path
2. Locate the bootstrap.py script
3. Generate a service configuration with correct paths
4. Install it in the platform-specific location

## Template Files

### Linux: `graphiti-bootstrap.service`
- **Generated by**: `SystemdServiceManager._create_service_unit()`
- **Installed to**: `~/.config/systemd/user/graphiti-bootstrap.service`
- **Platform**: Linux with systemd

### macOS: `com.graphiti.bootstrap.plist`
- **Generated by**: `LaunchdServiceManager._create_plist()`
- **Installed to**: `~/Library/LaunchAgents/com.graphiti.bootstrap.plist`
- **Platform**: macOS with launchd

### Windows: NSSM
- **No template file** - NSSM manages service configuration via registry
- **Managed by**: `WindowsServiceManager`
- **Platform**: Windows (requires NSSM installation)

## Manual Installation (Not Recommended)

If you need to manually install the service without using `graphiti-mcp daemon install`:

### Linux (systemd)
```bash
# 1. Copy template
cp graphiti-bootstrap.service ~/.config/systemd/user/

# 2. Edit paths in the file
nano ~/.config/systemd/user/graphiti-bootstrap.service

# 3. Install
systemctl --user daemon-reload
systemctl --user enable graphiti-bootstrap
systemctl --user start graphiti-bootstrap
```

### macOS (launchd)
```bash
# 1. Copy template
cp com.graphiti.bootstrap.plist ~/Library/LaunchAgents/

# 2. Edit paths in the file
nano ~/Library/LaunchAgents/com.graphiti.bootstrap.plist

# 3. Install
launchctl load ~/Library/LaunchAgents/com.graphiti.bootstrap.plist
```

### Windows (NSSM)
See: https://nssm.cc/usage

## Recommended: Use `graphiti-mcp daemon install`

The recommended installation method is:

```bash
graphiti-mcp daemon install
```

This will:
- Auto-detect your platform
- Generate the correct service configuration
- Install and start the service
- Handle all platform-specific details

See: `.claude/implementation/DAEMON_ARCHITECTURE_SPEC_v1.0.md`
