**Line 115-120** - Configuration Options description:
- `enabled` - Enable/disable session tracking (default: `false`)
- `watch_path` - Directory to monitor for JSONL files (default: `null`, uses `~/.claude/projects/`)
- `inactivity_timeout` - Seconds of inactivity before session is considered closed (default: 900)
- `check_interval` - Seconds between checks for file changes (default: 60)
