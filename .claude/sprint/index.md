## Sprint Goal

Remove deprecated session-based monitoring code that was superseded by the turn-based indexing architecture. The turn-based approach (implemented in `turn-based-preprocessing-injection-v1.0`) replaces time-based session detection with immediate call/response graph uploading, making `inactivity_timeout`, `check_interval`, `auto_summarize`, and related file watchers/periodic checkers obsolete.
