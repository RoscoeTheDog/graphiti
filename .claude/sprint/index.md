## Sprint Goal

Implement a dedicated virtual environment at `~/.graphiti/.venv/` for the daemon architecture, enabling portable installation that is decoupled from the source repository. Users can clone the repo, run an installer, and have CLI commands available globally without hardcoded paths or dependency conflicts.
